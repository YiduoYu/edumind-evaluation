# EduMind 学生心理自我评价系统

EduMind 是一个基于 Spring Boot 构建的后端服务系统，旨在通过学生提交的自我评价内容，结合 AI 情感分析与心理学模型，辅助教师及时发现学生可能存在的心理健康问题。

---

## 核心功能

-  学生自我评价提交接口
-  NLP 情绪分析（待集成）
-  自动预警系统：若学生连续两次提交负面情绪，将标记为预警，通知教师
-  教师端查看学生评价与历史数据
-  MySQL 数据库存储结构清晰，支持按学生ID检索记录
- Echarts 显示学生情绪波动图表
